## Prereqisites

* Java >= 1.8
* Maven >= 3.5.3
* yarn 
* angular cli

## Build

`mvn clean package`

## Run

`mvn clean spring-boot:run`

_Default port is_ **9999**

Point you browser to http://localhost:9999

### Run using 8080 port

`mvn clean spring-boot:run -Dserver.port=8080`

Point you browser to http://localhost:8080
